# 停车宝 - 拟人化汽车插图使用指南

## 概述

本文件夹包含停车宝产品所需的拟人化汽车插图，这些插图旨在更直观地传达产品价值主张：避免罚款、及时接收提醒和享受无忧驾驶体验。

## 插图风格指南

为确保视觉一致性，所有拟人化汽车插图应遵循以下设计风格：

1. **现代简约风格**：使用简洁的线条和形状，避免过于复杂的细节
2. **友好表情**：汽车前脸设计应包含友好、可信赖的表情元素
3. **品牌色系**：主要使用蓝色系（#3b82f6）作为主色调，辅以白色和浅灰色
4. **一致的透视角度**：保持45°角的3/4视图，便于展示汽车的表情和动作

## 推荐使用场景

### 1. 产品流程说明

在「使用方法」部分的各个步骤中，可使用不同表情的汽车插图来表达用户在各阶段的体验：

- **步骤1（订阅产品服务）**：展示期待表情的汽车
- **步骤2（输入Bark推送地址）**：展示专注表情的汽车
- **步骤3（生成快捷指令）**：展示思考/处理中表情的汽车
- **步骤4（使用快捷指令）**：展示满意/成功表情的汽车

### 2. 用户痛点与解决方案对比

在「产品特点」部分，可使用对比插图：

- **痛点场景**：展示焦虑/担忧表情的汽车（收到罚单）
- **解决方案**：展示放松/安心表情的汽车（使用停车宝后）

### 3. 订阅方案展示

在「订阅服务」部分的三种套餐中，可使用不同等级的汽车插图：

- **月度订阅**：基础款汽车插图
- **季度订阅**：升级款汽车插图（添加一些小装饰）
- **年度订阅**：豪华款汽车插图（添加VIP标识和更多装饰）

## 文件命名规范

所有插图文件应遵循以下命名规范：

```
car_[emotion]_[scene].svg
```

例如：
- car_happy_notification.svg
- car_worried_fine.svg
- car_relieved_solution.svg
